## ⚠️ COMMON ISSUES

### Issue: "500 Database Connection Error"
**Fix:** 
1. Check `DATABASE_URL` is set in Railway
2. Verify PostgreSQL service is running (Railway dashboard)
3. Check logs: `railway logs --service backend`

### Issue: "CORS Error / API 403"
**Fix:**
1. Check `ALLOWED_ORIGINS` includes your frontend domain
2. Restart backend service (Railway dashboard)
3. Check browser console for exact error

### Issue: "Frontend not loading / 404"
**Fix:**
1. Check frontend build succeeded (Railway logs)
2. Verify Procfile command: `npm run build && npx serve -s build`
3. Check `package.json` has correct scripts

### Issue: "API returns 401 Unauthorized"
**Fix:**
1. Frontend must pass JWT token in headers (useAuth hook)
2. Login first to get token
3. Check token not expired (useAuth manages this)
